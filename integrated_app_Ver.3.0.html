📤 출고 요청 등록
발주 필요 수량 자동계산
필요 수량을 입력하면 현재 재고를 차감한 값이 자동으로 계산됩니다. 재고가 충분하면 0으로 표시됩니다.
출고 완료 조건
납품 수량 ≥ 출고 요청 수량 AND 출고확인일이 입력되어야 출고 완료로 전환됩니다. 둘 중 하나라도 빠지면 일부 출고로 유지됩니다.
비고(S/N) 란
시리얼 넘버 또는 기타 메모를 자유롭게 입력할 수 있습니다.
📋 출고 요청 목록
행 색상 의미
초록 — 출고 완료
노랑 — 출고 기한 10일 이내 임박
연노랑 — 일부 출고
연빨강 — 미출고
납품 수량 입력 시 재고 자동 차감
편집 저장 시 납품 수량 증가분만큼 해당 품목 재고에서 자동으로 차감되고, 재고 관리의 입출고 기록에도 자동으로 남습니다.
발주필요 수량
출고 요청 수량이 현재 재고보다 많을 경우 그 차이가 빨간색으로 자동 표시됩니다.
출고 준비
실제 출고 전 준비된 수량을 직접 입력하는 칸입니다. 재고에 영향을 주지 않습니다.
📦 재고 관리
재고 상태 색상
충분 현재 재고 ≥ 최소 기준
주의 최소 기준의 30~100% 사이
부족 최소 기준의 30% 이하 또는 0
수량 조정(+/−) vs 입출고 등록 차이
수량 조정은 빠르게 수량만 바꿀 때 사용하며 "수량 직접 조정" 메모로 기록됩니다. 입출고 등록은 날짜, 프로젝트, 메모를 상세하게 남길 때 사용합니다.
입출고 기록 삭제 시
기록을 삭제하면 해당 수량이 재고에 자동으로 역산됩니다. (입고 기록 삭제 → 재고 차감 / 출고 기록 삭제 → 재고 복구)
카테고리 관리
우측 상단 카테고리 관리 버튼에서 카테고리 추가·수정·삭제가 가능합니다. 카테고리 삭제 시 해당 카테고리의 품목도 함께 삭제되니 주의하세요.
💾 데이터 저장
자동 저장
데이터 변경 후 3초가 지나면 구글 시트에 자동으로 저장됩니다. 우측 하단 토스트 알림으로 저장 상태를 확인할 수 있습니다.
저장 실패 시
네트워크 연결을 확인하고 페이지를 새로고침하세요. 저장되지 않은 변경사항은 새로고침 시 사라질 수 있습니다.
여러 명이 동시에 사용할 때
같은 항목을 동시에 수정하면 마지막에 저장한 내용이 반영됩니다. 중요한 데이터는 동시 수정을 피해주세요.
📅 연말 정리 방법
매년 12월 말 권장 작업 순서
1구글 시트 복사본 저장
구글 드라이브에서 현재 스프레드시트 우클릭 → 사본 만들기 → 이름을 2025 재고기록 으로 변경 후 보관
2입출고기록 탭 초기화
원본 시트의 입출고기록 탭 열기 → 전체 선택(Ctrl+A) → Delete로 내용 삭제
3완료된 출고요청 정리 (선택)
출고요청 탭에서 완료된 항목들도 정리하려면 같은 방법으로 삭제
4재고 실사 후 수량 업데이트
재고 관리에서 실제 재고와 다른 항목들 수량 조정
⚠️ 재고 데이터는 건드리지 않아도 됩니다. 입출고기록만 초기화하면 새해부터 깨끗하게 시작됩니다.
