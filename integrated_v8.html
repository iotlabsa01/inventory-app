대시보드 개요
요약 지표 (상단 4칸)
전체 재고 품목 수, 부족 품목 수, 미완료 출고요청 건수, 이번달 출고 완료 건수를 한눈에 보여줍니다.
경고 배너
출고 기한이 10일 이내로 임박한 요청과 마감일이 2주 이내인 현장이 자동으로 배너로 표시됩니다. 빨간색일수록 긴급합니다.
현장별 진행 현황 / 최근 입출고
등록된 현장의 출고 상태와 최근 입출고 기록을 빠르게 확인할 수 있습니다. 로그인 후 자동으로 최신 데이터를 불러옵니다.
출고 요청 등록
발주 필요 수량 자동 계산
필요 수량 입력 시 현재 재고를 자동 차감해 발주 필요 수량을 계산합니다. 재고가 충분하면 0으로 표시됩니다.
출고 완료 조건
납품 수량 ≥ 출고 요청 수량 AND 출고확인일이 입력되어야 출고 완료로 전환됩니다.
출고 요청 목록 — 행 색상 의미
초록 — 출고 완료
노랑 — 출고 기한 10일 이내 임박
연노랑 — 일부 출고
연빨강 — 미출고
납품 수량 저장 시 해당 수량만큼 재고에서 자동 차감되고 입출고 기록에도 자동으로 남습니다.
재고 관리
재고 상태 기준
충분 현재 재고 ≥ 최소 기준
주의 최소 기준의 30~100% 사이
부족 최소 기준의 30% 이하 또는 0
수량 조정 vs 입출고 등록 차이
수량 조정(+/−)은 빠른 수정용으로 "직접 조정" 메모로 기록됩니다. 입출고 등록은 날짜·프로젝트·메모를 상세하게 남길 때 사용합니다.
현장 등록 및 관리
현장 추가
고객사, 프로젝트명, 주소, 담당자, 설치 항목(난방/조명 등), 마감 일정을 입력해 현장을 등록합니다. 등록된 현장은 출고 요청 시 프로젝트로 연결됩니다.
마감일 경고
D-14 이내 카드 테두리 노랑 + 배너
D-7 이내 카드 테두리 빨강 + 배너
미완료 공정이 있을 때만 경고가 표시됩니다.
마감 현장 자동 이동
시공 내역 전체 완료 + 체크리스트 전체 체크 시 자동으로 "마감 현장"으로 이동됩니다. 필요 시 진행중으로 복원 가능합니다.

시스템 탭 (난방·조명·스위치 등)
현장에 등록된 시스템만 활성화됩니다. 비활성 탭은 흐리게 표시되며 클릭되지 않습니다.
공정률 계산 방식
시공 내역 완료 행 + 체크리스트 체크 항목을 합산해 전체 공정률(%)을 계산합니다. 100%가 되면 마감 현장으로 자동 이동됩니다.
체크리스트 항목 추가·수정·삭제
우측 상단 "+ 항목 추가" 버튼으로 단계(착공전/시공중/검수/인수인계)와 내용을 입력해 추가할 수 있습니다. 각 항목의 ✎·✕ 버튼으로 수정·삭제가 가능합니다.
저장 버튼을 눌러야 변경사항이 구글 시트에 반영됩니다. 탭 전환 시에는 현재 내용이 자동으로 임시 저장됩니다.
캘린더
스케줄 등록
날짜 칸을 클릭하면 스케줄 추가 모달이 열립니다. 내용과 등록자 이름을 입력해 등록하며, 최대 2건까지 미리보기로 표시됩니다.
스케줄 삭제
날짜 클릭 후 모달에서 항목 우측 × 버튼을 누르면 삭제됩니다. 캘린더 데이터는 현재 브라우저 세션 내에서만 유지됩니다.
구글 시트 자동 저장
자동 저장 방식
데이터 변경 후 3초가 지나면 구글 시트에 자동 저장됩니다. 우측 하단 토스트 알림으로 저장 상태를 확인할 수 있습니다.
저장되는 시트 탭
재고 · 입출고기록 · 출고요청 · 현장 · WBS 탭에 자동 저장됩니다. 사용자 계정은 사용자 탭에서 직접 관리합니다.
여러 명이 동시에 같은 항목을 수정하면 마지막에 저장한 내용이 반영됩니다. 중요한 데이터는 동시 수정을 피해주세요.
